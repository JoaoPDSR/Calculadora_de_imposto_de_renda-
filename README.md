# 💳 Finance Dashboard

Sistema financeiro com interface moderna inspirado em bancos digitais.

## 🚀 Funcionalidades

- Cálculo de imposto de renda
- Suporte a dependentes
- Calculadora de renda anual
- Dashboard com gráficos
- Histórico salvo em JSON

## 🛠 Tecnologias
- IA
- Python
- CustomTkinter
- Matplotlib

## ▶️ Como executar

```bash
pip install -r requirements.txt
python app.py
